预定义预设

预设 特点 推荐使用场景
Default - 最佳视觉感知
- 在精度、填充率、微小物体等方面的整体性能良好		 - 通用场景
- 机器人应用
Hand - 清晰的手部和手指边缘 - 手势识别
High Accuracy - 高置信度深度
- 几乎无噪声深度值
- 填充率较低		 - 避障
- 物体扫描
High Density - 更高填充率
- 更多微小物体
- 可能存在噪声深度值		 - 物体识别
- 抓取和放置
- 前景和背景动画
Medium Density - 填充率和精度的平衡性能
- 与 Default 相比：填充率较低，边缘质量更好		 - 通用场景和 Default 的替代方案
Custom - 用户自定义预设
- 从上述预设派生，具有自定义修改，例如后处理管道的新配置、深度 AE 功能的修改平均强度设定点等		 - 通过使用自定义配置可获得比预定义预设更好的深度性能
- 适用于已建立的自定义配置

根据您的具体使用场景选择合适的预设名称，并将其设置为 device_preset 参数的值。
